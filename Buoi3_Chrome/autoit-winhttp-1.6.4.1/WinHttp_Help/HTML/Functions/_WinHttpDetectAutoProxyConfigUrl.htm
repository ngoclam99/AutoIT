Function Reference

_WinHttpDetectAutoProxyConfigUrl

Finds the URL for the Proxy Auto-Configuration (PAC) file.

Syntax

#include "WinHttp.au3"
_WinHttpDetectAutoProxyConfigUrl($iAutoDetectFlags)

Parameters

$iAutoDetectFlags Specifies what protocols to use to locate the PAC file.

Return Value

Success - Returns URL for the PAC file.
Failure - Returns empty string and sets @error:
                    1 - DllCall failed
                    2 - Internal failure.

Remarks

$iAutoDetectFlags values are defined in WinHttpconstants.au3


Related

_WinHttpGetDefaultProxyConfiguration, _WinHttpGetIEProxyConfigForCurrentUser, _WinHttpSetDefaultProxyConfiguration

Example

Copy to clipboard




#include "WinHttp.au3"

Opt("MustDeclareVars", 1)

; This could take some time
Global $sPACurl = _WinHttpDetectAutoProxyConfigUrl(BitOR($WINHTTP_AUTO_DETECT_TYPE_DHCP, $WINHTTP_AUTO_DETECT_TYPE_DNS_A))

MsgBox(64, "_WinHttpDetectAutoProxyConfigUrl", "The URL for the Proxy Auto-Configuration (PAC) file is: " & $sPACurl)
