Function Reference

_WinHttpCrackUrl

Separates a URL into its component parts such as host name and path.

Syntax

#include "WinHttp.au3"
_WinHttpCrackUrl($sURL [, $iFlag = Default ])

Parameters

$sURL String. Canonical URL to separate.
$iFlag [optional] Flag that control the operation. Default is $ICU_ESCAPE

Return Value

Success - Returns array with 8 elements:
                    $array[0] - scheme name
                    $array[1] - internet protocol scheme
                    $array[2] - host name
                    $array[3] - port number
                    $array[4] - user name
                    $array[5] - password
                    $array[6] - URL path
                    $array[7] - extra information
Failure - Returns 0 and sets @error:
                    1 - DllCall failed

Remarks

$iFlag is defined in WinHttpConstants.au3 and can be:
       $ICU_DECODE - Converts characters that are "escape encoded" (%xx) to their non-escaped form.
       $ICU_ESCAPE - Escapes certain characters to their escape sequences (%xx).
