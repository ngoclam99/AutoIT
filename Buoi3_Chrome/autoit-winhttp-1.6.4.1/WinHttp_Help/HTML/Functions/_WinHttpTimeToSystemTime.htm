Function Reference

_WinHttpTimeToSystemTime

Takes an HTTP time/date string and converts it to array (SYSTEMTIME structure values).

Syntax

#include "WinHttp.au3"
_WinHttpTimeToSystemTime($sHttpTime)

Parameters

$sHttpTime Date/time string to convert.

Return Value

Success - Returns array with 8 elements:
                    $array[0] - Year,
                    $array[1] - Month,
                    $array[2] - DayOfWeek,
                    $array[3] - Day,
                    $array[4] - Hour,
                    $array[5] - Minute,
                    $array[6] - Second.,
                    $array[7] - Milliseconds.
Failure - Returns 0 and sets @error:
                    1 - DllCall failed
