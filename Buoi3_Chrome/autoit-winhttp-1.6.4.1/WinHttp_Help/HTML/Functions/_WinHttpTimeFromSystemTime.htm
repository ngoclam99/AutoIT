Function Reference

_WinHttpTimeFromSystemTime

Formats a system date and time according to the HTTP version 1.0 specification.

Syntax

#include "WinHttp.au3"
_WinHttpTimeFromSystemTime()

Parameters

 None.


Return Value

Success - Returns time string.
Failure - Returns empty string and sets @error:
                    1 - Initial DllCall failed
                    2 - Main DllCall failed

Related

_WinHttpTimeToSystemTime

Copy to clipboard




#include "WinHttp.au3"

Opt("MustDeclareVars", 1)

MsgBox(64, "WinHttp time", _WinHttpTimeFromSystemTime())
